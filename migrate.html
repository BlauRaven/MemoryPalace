🔄 Migrate Experiences to Firestore

This tool will migrate all your experiences from experiences.json to Firebase Cloud Firestore.

Before you start:

  • Make sure your Firestore database is created in Firebase Console
  • Security rules are set to allow read/write access
  • Your experiences.json file is in the same directory

After migration completes successfully, your experiences will be available in Firestore. You can then use the main index.html to view and add new experiences!
